# Рэй Desktop

**Рэй** — персональный AI-помощник для общения, целей, учебы, памяти и фокуса.

Десктопное приложение на Electron, которое объединяет веб-версию Рэя с возможностями нативного приложения: системный трей, глобальное меню, автообновления и работа в фоне.

### Из исходников

```bash
# Клонировать репозиторий
git clone https://github.com/Kaimo-ki/ray-desktop.git
cd ray-desktop

# Установить зависимости
npm install

# Скопировать веб-сайт в папку ray_site
cp -r /путь/к/ray_site ./

# Запустить в режиме разработки
npm start

# Собрать приложение
npm run build:win   # Windows
npm run build:mac   # macOS
npm run build:linux # Linux
🚀 Возможности

    Чат с Рэем — задавайте вопросы, делитесь мыслями, ставьте цели

    Голосовой ввод — говорите вместо печати (поддержка Speech Recognition)

    Озвучивание ответов — Рэй может отвечать голосом

    Память — Рэй запоминает важные факты и стиль общения

    Цели и планы — большие задачи разбиваются на маленькие шаги

    Спутник — плавающая иконка Рэя поверх всех окон (настраивается в приватности)

    Системный трей — иконка в трее для быстрого доступа

    Автообновления — приложение обновляется автоматически

    Одиночный экземпляр — только одно окно приложения

    Сворачивание в трей — закрытие сворачивает, а не завершает приложение

📁 Структура проекта
ray-desktop/
├── main.js                 # Главный процесс Electron
├── preload.js              # Изолированный мост для IPC
├── renderer.js             # Загрузчик веб-приложения
├── index.html              # Загрузочный экран
├── package.json            # Конфигурация и зависимости
├── icons/                  # Иконки для всех платформ
│   ├── icon.png            # 512x512
│   ├── icon.ico            # Windows
│   └── icon.icns           # macOS
└── ray_site/               # Веб-приложение (копируется)
    ├── index.html
    ├── config.js
    ├── script.js
    ├── styles.css
    ├── sw.js
    ├── offline.html
    └── assets/

🗝️ Клавиши
Комбинация	Действие
Cmd/Ctrl + N	Новое окно
Cmd/Ctrl + R	Перезагрузить
Cmd/Ctrl + Shift + I	Инструменты разработчика
Cmd/Ctrl + +	Увеличить масштаб
Cmd/Ctrl + -	Уменьшить масштаб
Cmd/Ctrl + 0	Сбросить масштаб
F11 / Cmd+Ctrl+F	Полноэкранный режим

🔒 Приватность

    Рэй не читает ваш компьютер, телефон, файлы или другие приложения

    Рэй видит только сообщения и файлы, которые вы отправляете напрямую

    Память и спутник включаются только с вашего разрешения

    Если выбрано "Только чат", Рэй работает без памяти и спутника
