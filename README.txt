Ray website

Как открыть:
1. Открой index.html в браузере.
2. Папку можно загрузить на GitHub Pages, Netlify, Cloudflare Pages или любой статический хостинг.

Файлы:
- index.html - страница сайта
- styles.css - стили
- script.js - демо-чат
- assets/ray-hero.png - hero-изображение

Важно:
Демо-чат на сайте пока статический. Он показывает поведение интерфейса, но не подключен к настоящему Telegram-боту или AI API.
