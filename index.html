Чат: вопрос, мысль, цель или просьба.

Telegram: после связки бот читает ту же память.

Что уже есть

Это уже не просто обложка: сайт разговаривает с backend.

Ray Web отправляет сообщения в Railway API, API пишет память в PostgreSQL, а Telegram-бот может читать ту же память через связку.

Слабые места сейчас: голос в web, фото-анализ, красивый мобильный поток входа, трекер целей и полноценный desktop companion.

Google

Главный быстрый вход. После Google человек возвращается на сайт уже с сессией.

Email + пароль

Запасной вход, если человек не хочет Google.

Telegram link

Код с сайта отправляется боту, после этого память становится общей.

SMS позже

Нужен отдельный платный SMS-провайдер, поэтому пока не основной путь.
