Как это будет работать

Telegram и сайт подключаются к одному Рэю.

01 Telegram

Основной канал: голос, стикеры, цели, память.

02 Ray API

Общая логика и память для всех интерфейсов.

03 Web

Этот сайт станет настоящим чатом после VM.

Следующий шаг

Поднять backend, вставить URL в config.js — и сайт оживёт.

Пока VM готовит друг, сайт работает как демо-интерфейс. После запуска API мы подключим `window.RAY_API_URL`, и web-чат будет использовать ту же память, что Telegram.
